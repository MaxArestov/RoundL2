Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
# Ответ:

Программа выведет числа от 0 до 9, каждое на новой строке. 
Однако после этого произойдет deadlock, так как цикл `for range` 
используемый с каналом `ch` ожидает дополнительные значения, 
и канал никогда не закрывается.

Чтобы исправить программу и не допустить дедлок, 
надо добавить `close(ch)` в конец анонимной горутины.

6

8

fatal error: all goroutines are asleep - deadlock!    
```
