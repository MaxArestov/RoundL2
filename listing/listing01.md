# Программа на Go

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```
Ответ:
```
Программа выводит срез массива `a`, начиная с элемента с индексом 1 
и заканчивая элементом с индексом 3 (не включая 4й). 
В результате, вывод программы будет следующим:
[77 78 79]
```
